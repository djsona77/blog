클라우드 서버가 무엇일까?

물리적인 큰 서버를 여러개로 나눠서 가상 서버로 사용하는 방법
예를들어 MS, AMAZON이 사용하고 있다

서버와 클라우드의 차이점

서버 : 서버주인이 서버를 관리
장점 : 보안이 강력
단점 : 비싸고, 만들때 오래걸림
클라우드 : 가상서버를 만들고 바로 사용
장점 : 만들고 바로 사용
장점 : 유동성이 좋음
장점 : 속도도 빠름
단점 : 운영 비용이 비쌈

해킹 사례

CAPITAL ONE 회사의 클라우드가 해킹당함
